## Format

Blog posts are written in Markdown format for easy reading and version control.

### File Naming Convention

Use the following format:
- `YYYY-MM-DD-title-of-post.md`

Example:
- `2026-02-09-kubernetes-deployment-strategies.md`
- `2026-01-15-serverless-architecture-aws.md`

## Adding Blog Posts

1. Create a new Markdown file following the naming convention
2. Include front matter with metadata (title, date, author, tags)
3. Write your content using Markdown
4. Update the index below
5. Submit a pull request

## Blog Post Template
