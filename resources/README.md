# Resources

This directory contains additional learning resources, references, tools, and supplementary materials.

## Content Types

- **Reference Guides**: Quick reference sheets and cheat sheets
- **Tools & Scripts**: Utility scripts and automation tools
- **External Links**: Curated lists of external resources
- **Books & Papers**: Recommended reading materials
- **Certification Guides**: Study materials for certifications

## Organization

Resources are organized by category:
- `cheatsheets/` - Quick reference guides
- `tools/` - Scripts and utilities
- `links/` - Curated external resources
- `certifications/` - Certification study guides

## Adding Resources

1. Choose the appropriate category folder
2. Add your resource with clear documentation
3. Update the index below
4. Submit a pull request
