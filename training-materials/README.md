# Training Materials

This directory contains hands-on training materials, labs, tutorials, and course content.

## Content Types

- **Labs**: Step-by-step hands-on exercises
- **Tutorials**: Guided learning content
- **Course Materials**: Complete course content and syllabi
- **Exercises**: Practice problems and solutions
- **Code Samples**: Example implementations

## Organization

Materials are organized by topic:
- `devops/` - DevOps practices and tools
- `cloud/` - Cloud platform tutorials (AWS, Azure, GCP)
- `ai-ml/` - Artificial Intelligence and Machine Learning
- `security/` - Security best practices
- `infrastructure/` - Infrastructure as Code and management

## Adding Training Materials

1. Create a folder for your topic if it doesn't exist
2. Include a README.md with overview and prerequisites
3. Add your training materials with clear documentation
4. Update the index below
