# Presentations

This directory contains training presentation decks (PPT/PPTX files) covering various topics.

## Categories

- DevOps
- Cloud Computing (AWS, Azure, GCP)
- Artificial Intelligence
- Security
- Infrastructure

## File Naming Convention

Please use descriptive names for presentations:
- `devops-cicd-fundamentals.pptx`
- `aws-serverless-architecture.pptx`
- `kubernetes-advanced-topics.pptx`

## Adding Presentations

1. Upload your presentation file to the appropriate category folder (or create one if needed)
2. Update the index in this README with a link and brief description
3. Submit a pull request
